# NHKラジオ ダウンローダー

このスクリプトは、NHKラジオの語学番組をダウンロードするためのPythonスクリプトです。
Seleniumを使用してNHKのオンデマンドページから音声ファイルのURLを取得し、ffmpegを使用してダウンロードします。

## 対象番組
一度の実行で以下の4つの番組の最新の放送分をダウンロードします。

- ラジオ英会話
- 英会話タイムトライアル
- 現代英語
- ラジオビジネス英語

ファイル名は `番組名_日付.mp3` （例：`ラジオ英会話_4月1日.mp3`）のように保存されます。

## 必要な環境・依存関係

このスクリプトを実行するためには、以下のソフトウェアがインストールされている必要があります。

1. **Python 3.x**
2. **Google Chrome** ブラウザ
3. **ffmpeg**（コマンドラインから実行できるようにパスが通っていること）

### Pythonパッケージのインストール

必要なライブラリをインストールします。

```bash
pip install requests selenium
```

## 使い方

以下のコマンドを実行するだけで、設定された番組の音声がダウンロードされます。

ダウンロードされたmp3ファイルは、`main.py` を実行したフォルダ（カレントディレクトリ）に保存されます。

```bash
python main.py
```

### ダウンロードの仕組みと最適化について
- **Selenium**: ブラウザを自動操作して、各番組のページからストリーミング再生用URL（`.m3u8`）を取得しています（バックグラウンドで動作するヘッドレスモードを使用）。
- **ffmpeg**: `.m3u8` のURLから音声をダウンロードし、mp3ファイルとして保存します。
  - **最適化**: ネットワークの一時的な切断や遅延でダウンロードが途中で止まってしまったり、音飛びが発生するのを防ぐため、再接続オプション（`-reconnect` 等）を設定しています。

## トラブルシューティング
- **`ffmpeg: command not found` と表示される場合**
  ffmpegがインストールされていないか、システムの環境変数PATHに追加されていません。ffmpegをインストールしてパスを通してください。
- **ブラウザ関連のエラーが出る場合**
  Google Chromeがインストールされていることを確認してください。
